# MBPO PyTorch
A PyTorch reimplementation of MBPO (When to trust your model: model-based policy optimization)

# Dependency

Please refer to ./requirements.txt.

# Usage

    python ./mbpo/scripts/run_mbpo.py
    
  hyperparams in ./configs/mbpo.yaml
